# 🚀 SyncSpace — Real-Time Collaboration Platform

## 🌟 Overview

SyncSpace is a full-stack real-time collaboration platform designed to help teams manage workspaces, tasks, and communication in one unified system.

It provides a smooth workflow starting from user authentication to real-time collaboration inside workspaces.

---

## ✨ Core Features

### 🔐 Authentication System

* User Registration (Sign Up)
* Secure Login
* Session-based access
* Clean UI flow

---

### 🧩 Dashboard

* Displays system insights:

  * Total messages
  * Tasks completed
  * Team members
  * System uptime
* Quick action panel:

  * Create workspace
  * Add task
  * Invite user

---

### 🏢 Workspace Management

* Create new workspaces
* View all workspaces
* Switch between workspaces easily

---

### 📌 Task Management (Kanban Board)

* Task stages:

  * To Do
  * In Progress
  * Done
* Move tasks between stages
* Track task updates

---

### 💬 Real-Time Chat

* Instant messaging within workspace
* Live updates using WebSockets
* Activity tracking

---

### ✍️ Editor & Notes

* Write and edit content
* Live preview functionality
* Clean writing interface

---

### 👥 Team Collaboration

* Invite members to workspace
* Manage team members
* Activity log updates

---

### ⚡ Real-Time System

* Built with WebSockets
* Instant UI updates across users
* Backend-driven events

---

## 🏗️ Tech Stack

**Frontend**

* React + TypeScript
* Tailwind CSS
* Vite

**Backend**

* NestJS
* Prisma ORM
* WebSockets

**Database**

* SQLite (Development)

---

## 📁 Project Structure

```
SyncSpace/
├── backend/
├── frontend/
```

---

## ⚙️ Run Locally

```bash
git clone https://github.com/Kivaane/SyncSpace.git
cd SyncSpace
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
